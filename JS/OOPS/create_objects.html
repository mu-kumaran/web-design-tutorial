Creating objects using function and constructor

In console,
create objects using function by enter the code below :
const obj_manoj = createObjectUsingfunction("manoj");
obj_manoj.name;
obj_manoj.introduceSelf()

create objects using constructor by enter the code below:
const obj_muthu =   new   createObjectUsingConstructor("muthu");
obj_muthu.name;
obj_muthu.introduceSelf();
